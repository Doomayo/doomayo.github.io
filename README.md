# Waheed Abdulbaki Shurbaji - Portfolio

## Overview
This is a personal portfolio website for Waheed Abdulbaki Shurbaji, a Medical Informatics student at the University of Lübeck. The website showcases skills, experience, and projects in a modern, responsive interface.

## Features
-   **Multi-language Support**: Fully localized in German (DE), English (EN), and Arabic (AR) with RTL support.
-   **Dark/Light Mode**: Toggleable theme with persistent preference storage.
-   **Responsive Design**: Optimized for mobile, tablet, and desktop viewports using Tailwind CSS.
-   **Interactive Elements**: Smooth scrolling, animations, glassmorphism effects, and modal views for project details and contact information.

## Tech Stack
-   **HTML5**: Semantic structure.
-   **Tailwind CSS**: Utility-first CSS framework for styling.
-   **JavaScript (Vanilla)**: Logic for translations, theme toggling, and interactivity.
-   **FontAwesome**: Icons.
-   **Google Fonts**: "Outfit" typeface.

## Project Structure
-   `index.html`: The main entry point containing all structure, styles, and logic.
-   `IMG_1039.jpeg`: Personal profile image.

## Setup
Simply open `index.html` in any modern web browser to view the portfolio. No build step is required as Tailwind is loaded via CDN for this standalone version.
